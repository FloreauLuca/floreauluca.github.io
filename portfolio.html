School Projects

Minecraft-like

Program made on Jul 24, 2020

- Our Minecraft-like is a project where you can move inside an infinite generate world.

- This project was create in 2 months as part of the module of computer graphics. I work in team with 7 of my classmates in C++ using the NekoEngine. The NekoEngine is a custom C++ Engine using SDL 2.0 and OpenGL ES 3.0.
- I enjoy working on chunk generation and chunk rendering.

Checkout the release : github.com/LoshkinOleg/NekoEngine/releases

Ultra PingBong 3D

Game made on Nov 29, 2019

- Ultra PingBong 3D is an online multiplayer game in real-time.

- This project was created to learn the networking in Unity using Photon. It can be play with 2 to 16 players.

Checkout the release : github.com/FloreauLuca/PingBongBR/releases

Penguin Collector

Game made on Apr 16, 2019

- Penguin Collector is a shooter in which you must save the penguins from their predators.

- This project was created to learn the procedural generation and implementation of artificial intelligence. It is also available on mobile. I took care of the graphics, gameplay, and game design.

On The Dancefloor

Game made on Dec 18, 2018

- On The Dancefloor is a rhythm game in which you have to tap in rhythm with the music to make the dancer move.

- This project was created in one week as part of my formation at SAE. It was one of my first group projects. It's available on mobile.

Potion Cooking

Game made on Nov 25, 2018

- Potion Cooking is a point'n'click in which you play as a mage who must make and sell magic potions to his customers.

- This project was the first game created during my training at SAE. It allowed me to put into practice what I was learning and to create my first mobile game. Furthermore, I enjoyed creating the sprites with this minimalist style.
